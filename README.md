## 📖 Overview
In the fields of Humanities and Social Sciences, researchers often deal with highly unstructured and messy datasets. This project provides a **seamless interface** to bridge the gap between complex data manipulation and intuitive human language. By leveraging **Gemini 2.5 Flash**, the Agent interprets natural language instructions and executes robust Python logic in real-time.

## ✨ Key Features
* **Natural Language Interaction**: Perform complex tasks like *"Unify date formats"* or *"Filter invalid emails"* by simply typing.
* **Gemini 2.5 Flash Integration**: Powered by the latest 2026 multimodal model with advanced reasoning for structural data challenges.
* **Transparent Execution**: Users can expand the **"View Execution Logic"** tab to inspect the actual Pandas code generated by the AI, ensuring academic rigor and reproducibility.
* **Multi-format Support**: Native support for uploading and exporting data in `.csv`, `.xlsx`, and `.json` formats.
* **User-Friendly Dashboard**: Professional UI built with Streamlit for an effortless user experience.

## 🛠️ Technical Stack
* **LLM Core**: Google Gemini 2.5 Flash (via `google-generativeai`)
* **Frontend**: Streamlit
* **Data Science**: Python Pandas, Regex, OpenPyXL, XlsxWriter
* **Deployment**: Streamlit Cloud + GitHub Secrets Management

## 🚀 How to Use
1.  **Access**: Open the [Live Web App](https://ttu-datacleaningagent-hdew2mkgaqvsbjas8faqha.streamlit.app).
2.  **Upload**: Drag and drop your messy CSV or Excel file into the sidebar.
3.  **Command**: Type your cleaning requirements in the chat box (e.g., *"Remove duplicate entries based on the 'User_ID' column"*).
4.  **Verify & Export**: Preview the data, check the generated code, and download the refined version.

## 📝 Example Use Cases
* **Social Sciences**: Standardizing survey responses and handling messy categorical data.
* **Journalism**: Cleaning scraped web data and normalizing diverse timestamp formats.
* **Education**: Cleaning institutional records for longitudinal statistical analysis.
