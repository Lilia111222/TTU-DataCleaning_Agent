## 🤖 AI-Powered Data Cleaning Agent

An intelligent, conversational data processing assistant designed to bridge the gap between complex data manipulation and intuitive human language. Powered by the state-of-the-art **Gemini 2.5 Flash** model (2026 Edition).

---

## 📖 Overview
This project provides a seamless interface for researchers, journalists, and students to clean and transform messy datasets without writing a single line of code. By leveraging LLM-based code generation, the Agent interprets natural language instructions and executes robust Python logic in real-time.

 ✨ Key Features
1. Natural Language Interaction: Perform complex tasks like "Unify date formats" or "Filter invalid emails" simply by typing.
2. Gemini 2.5 Flash Integration: Utilizes the latest multimodal model with deep reasoning capabilities to handle structural data challenges.
3. Transparent Execution: Users can expand the "Execution Logic" tab to inspect the actual Pandas code generated by the AI.
4. Multi-format Support: Upload and export data in `.csv`, `.xlsx`, and `.json` formats.
5. User-Friendly Dashboard: Built with Streamlit for a clean, responsive, and professional UI.

 🛠️ Technical Stack
1.Core Model: Google Gemini 2.5 Flash (via `google-generativeai`)
2.Frontend: Streamlit
3.Data Processing: Python Pandas, Regex, OpenPyXL
4.Deployment: Streamlit Cloud + GitHub Secrets Management
